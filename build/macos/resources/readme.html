Quick start

  1. Launch OpenBox.app from Applications or Spotlight.
  2. Pick a language in the top-right toolbar (中文 / English).
  3. Switch between the Compress and Extract tabs at the top.

Supported formats

Format Compress Extract Notes
.zip✅✅5 compression levels
.tar✅✅
.tar.gz✅✅
.7z✅*✅*requires brew install 7zip

CLI

OpenBox also understands command-line flags:

openbox <archive>            # extract tab pre-filled
openbox -c <file|dir>...     # compress tab pre-filled
openbox -x <archive>          # alias for plain <archive>
openbox -version

Full documentation: github.com/samaidev/openbox
