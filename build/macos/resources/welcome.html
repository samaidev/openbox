Welcome to OpenBox

Open-source, free, cross-platform bilingual archiver — MIT licensed.

This installer will copy OpenBox.app to your Applications folder and register it as a viewer for the following archive types:

  • .zip  .tar  .tar.gz  .tgz
  • .7z  .rar  .iso

After installation, double-click any of those files to open them in OpenBox. You can also right-click an archive and choose Open With → OpenBox.

Compressing files

Drag files or folders into the Compress tab, pick a format, and click Compress. To create .7z archives, install the 7-Zip CLI first:

brew install 7zip

Uninstall

Drag OpenBox.app from Applications to the Trash.
